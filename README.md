# Asteroids (Summer 2021)

Based on the Atari arcade game

## Installation

- Windows: Just download and run `asteroids.exe`
- Other: Download `src` and run `asteroids.py`

## Controls
- `W ↑` Thrust
- `A ←` `D →` Rotate left/right
- `S ↓` Hyperspace
- `M` Mouse controls
- `Z` Hitboxes
- `X` Asteroid count
