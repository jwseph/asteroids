# Asteroids (Summer 2021)

Based on the Atari arcade game

## Installation

- Windows: Just download and run `asteroids.exe`
- Others:
  - Download `src` and run `asteroids.py`
  - To install pygame: `pip install pygame`

## Controls
- `W ↑` Thrust
- `A ←` `D →` Rotate left/right
- `S ↓` Hyperspace
- `M` Mouse controls
- `Z` Hitboxes
- `X` Asteroid count
- `Esc` Pause game
